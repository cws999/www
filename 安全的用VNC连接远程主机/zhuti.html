VNC是一个连接系统,允许您使用键盘和鼠标与远程服务器上的图形桌面环境进行交互 对于不熟悉命令行的用户,它使在远程服务器上管理文件,软件和设置更加容易.以下 操作将在devuan,debian,Ubuntu远程服务器上设置VNC服务器,并通过SSH隧道安全 地连接到该服务器
  1. 更新软件源
    2. apt update
  2. 安装必要的软件
    3. apt install openbox qterminal tightvncserver
  3. 开启vnc服务
    4. vncserver
  4. 设置密码
    5. Password: Verify:
  5. 无说明
    6. Would you like to enter a view-only password (y/n)? n
  6. 以下操作在本地主机进行.使用SSH安全转发到VNC的连接
    7. ssh -L 5901:127.0.0.1:5901 -C -N -l user-name your_server_ip
  7. 安装vnc软件
    8. apt install gvncviewer
  8. 开始连接,弹出一个窗口，输入第４步密码
    9. gvncviewer your_server_ip:1
